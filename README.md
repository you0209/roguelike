# DUNGEON QUEST

**ローグライク × JRPG** のブラウザゲーム。純粋な HTML / CSS / JS で動作し、サーバー不要で `index.html` をブラウザで開くだけでプレイできます。

---

## ゲームの概要

4つの階層からなるダンジョンを攻略し、最深部に潜む龍神を討伐することが目的です。
限られた回復手段の中でリソースを管理しながら進む、高難度なローグライク体験を目指しています。

---

## ゲームシステム

### 階層構成

| 階層 | 内容 |
|---|---|
| 第1〜3階層 | 雑魚戦・イベント・階段の3択から選んで進む |
| 第4階層 | ボス戦のみ（龍神ヴァルムドラグ） |

- **階段**を選択することで次の階層へ進める
- 階層移動のタイミングで **HP・MP が全回復**
- 階層移動後に **ショップ** でアイテム・装備を購入できる

### 戦闘システム（コマンドバトル）

| コマンド | 説明 |
|---|---|
| 攻撃 | 通常物理攻撃 |
| 防御 | そのターンのダメージを大幅軽減 |
| スキル | MP消費で強力な技を使用 |
| アイテム | 所持しているアイテムを使用 |

- ターン制（先攻はスピードに依存）
- 同じ階層で戦闘を重ねるほど敵が強くなる（HP +12% / ATK +6% / 戦闘ごと）
- **回復手段はほぼなし**。階層移動時の全回復が唯一の主要な回復機会

### ショップ

| カテゴリ | 内容 |
|---|---|
| 武器 | 攻撃力アップ |
| 防具 | 防御力・MP上限アップ |
| スキル | 新しいスキルを習得 |
| アイテム | 消費アイテム（ポーション・爆弾・毒薬など） |

購入には**戦闘報酬で得たゴールド**を使用します。

## スコア計算

```
スコア = (到達階層 - 1) × 600 + 総獲得ゴールド × 2 + 総戦闘回数 × 60
ボスクリア時 +5000 ボーナス
```

## 技術スタック

- **言語**: HTML / CSS / JavaScript（ライブラリなし）
- **描画**: Canvas 2D API（ドット絵スプライトは手描き）
- **対応環境**: モダンブラウザ（Chrome / Firefox / Safari / Edge）

## ファイル構成

```
rogulike/
├── index.html           # 全シーンのHTML構造・スクリプト読み込み

│   └── style.css        # ダークピクセルアートUI・全画面レイアウト
├── data/

│   ├── skills.js        # スキル定義・ショップ販売スキル一覧

    ├── game.js          # ゲームコア・シーン管理・共有ステート
    ├── battle.js        # バトルロジック・Canvasスプライト描画
    ├── shop.js          # ショップ処理・購入ロジック
    └── events.js        # イベントデータ・処理（実装予定）
```

## ファイル詳細

### `index.html`
ゲーム全体のHTML骨格。タイトル・バトル・ショップ・イベント・階段・ゲームオーバー・クリアの各シーンを `<div id="scene-xxx">` で管理し、`hidden` クラスで表示切り替えを行う。スクリプトの読み込み順（`data/` → `js/`）もここで管理。

### `css/style.css`
全画面のスタイル定義。ダークなピクセルアート風UIをCSSのみで実現。HPバー・コマンドボタン・ショップタブ・階段確認画面などのレイアウトを含む。

---

### `js/game.js` ― ゲームコア（共通）

| 要素 | 説明 |
|---|---|
| `GS` オブジェクト | 全ファイルから参照する共有ゲームステート（HP・MP・ゴールド・装備など） |
| `resetGame()` | ゲーム開始時の初期化 |
| `showScene(name)` | シーン切り替え |
| `initFloorSelect()` | 戦闘・イベント・階段をランダム選択して遷移 |
| `descendFloor()` | 階層移動・HP/MP全回復・ショップへ |
| `gotoGameOver()` / `gotoVictory()` | 終了画面の表示 |
| `pct()` / `makeMenuBtn()` | 全ファイルから使うユーティリティ関数 |

### `js/battle.js` ― バトルロジック（担当B）

| 要素 | 説明 |
|---|---|
| `spawnEnemy()` | 階層・戦闘回数に応じてスケールした敵を生成 |
| `initBattle()` | バトル画面の初期化 |
| `doAttack()` / `doDefend()` / `doSkill()` / `doItem()` | プレイヤーの各行動 |
| `doEnemyTurn()` | 敵のターン処理（ボスは専用スキルあり） |
| `checkWin()` / `checkLose()` | 勝敗判定・報酬付与 |
| `calcPhysDmg()` / `calcMagicDmg()` | ダメージ計算式 |
| `startBattleAnim()` | requestAnimationFrame によるアニメーションループ |
| `drawPlayer()` / `drawEnemy()` など | Canvas手描きスプライト（プレイヤー＋11種の敵） |

### `js/shop.js` ― ショップ処理（担当A）

| 要素 | 説明 |
|---|---|
| `initShop()` / `renderShop()` | ショップ画面の表示・タブ切り替え |
| `isOwned()` | 購入済みアイテムの判定 |
| `doBuy()` | 購入処理（武器・防具・スキル・アイテムで処理が異なる） |

### `js/events.js` ― イベント処理（担当C）

現在はプレースホルダー。`EVENTS` 配列にイベントデータを追加していく予定。

---

### `data/enemies.js` ― 敵データ（担当B）

| 要素 | 説明 |
|---|---|
| `ENEMIES` | 10種の敵データ。HP・攻撃・防御・報酬ゴールド・スプライト名などを定義 |
| `FLOOR_ENEMIES` | 階層ごとの出現敵リスト（1階: スライム系〜4階: ドラゴン固定） |

### `data/skills.js` ― スキルデータ（担当C）

| 要素 | 説明 |
|---|---|
| `SKILLS` | 6種のスキル定義。MPコスト・タイプ（物理/魔法/回復/バフ）・威力など |
| `SHOP_SKILLS` | ショップで販売するスキルと価格（`SKILLS` を参照して価格を付加） |

### `data/items.js` ― アイテムデータ（担当B）

6種の消耗品定義（ポーション・ハイポーション・エーテル・爆弾・毒の小瓶・蘇生草）。`type` フィールドでバトル中の使用効果が変わる。

### `data/relics.js` ― 武器・防具データ（担当A）

武器4種・防具4種の定義。攻撃/防御ボーナス・MPボーナス・価格を持つ。**レリックシステムへの改変予定**。
