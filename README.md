# DUNGEON QUEST

**ローグライク × JRPG** のブラウザゲーム。
純粋な HTML / CSS / JS で動作し、サーバー不要で `index.html` をブラウザで開くだけでプレイできます。

---

## ゲームの概要

7つの階層からなるダンジョンを攻略し、最深部に潜む龍神を討伐することが目的です。
限られた回復手段の中でリソースを管理しながら進む、高難度なローグライク体験を目指しています。

---

## 遊び方

### 基本の流れ

各階層で **戦闘・イベント・階段** の3択が提示されます。
階段を選ぶと次の階層へ進み、HP・MP が全回復します。
階層移動後は **ショップ** でレリックやアイテムを購入できます。

### 戦闘コマンド

| コマンド | 説明 |
|---|---|
| 攻撃 | 通常物理攻撃 |
| 防御 | そのターンのダメージを大幅軽減 |
| スキル | MP消費で強力な技を使用 |
| アイテム | 所持しているアイテムを使用 |

- ターン制（プレイヤー先攻）
- 同じ階層に長く留まるほど敵が強くなっていく

### レリック

ショップで購入できるアイテム。所持しているだけで効果が発動し、ビルドの方向性が変わります。
組み合わせ次第でさまざまな戦い方が生まれます。

---

## 技術スタック

- **言語**: HTML / CSS / JavaScript（ライブラリ・フレームワークなし）
- **描画**: Canvas 2D API（スプライトは手描き）
- **対応環境**: モダンブラウザ（Chrome / Firefox / Safari / Edge）

---

## ローカルで動かす

```bash
git clone https://github.com/you0209/roguelike.git
cd roguelike
# index.html をブラウザで開くだけ
open index.html
```
