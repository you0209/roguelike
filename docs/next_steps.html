2. 今後チームで話し合うべき議題

⚠ 以下はすべて 未決定 の議題です。まず何から話し合うかの優先度を揃えることから始めてください。

① ゲームの方向性・コンセプト

議題 選択肢・論点 優先度
今後のジャンル定義 現状の「ソロ・レリックビルド型」を深掘りする vs「パーティ編成型ローグライト」へ拡張する 高
トルネコ・シレンとの差別化ポイント 「何が違うゲームか」を一言で言えるようにしておく（対外説明・設計判断の軸になる） 高
ターゲットユーザー ローグライク経験者向け？JRPG経験者向け？ライトゲーマー向け？ 中

② 仲間システム

議題 選択肢・論点 優先度
仲間システムを実装するか 入れる / 入れない / 将来的に入れる（まず現状を磨く） 高
パーティ人数 ソロ継続 / 2人 / 最大3人など 高
仲間の役割 バトルに直接参加（毎ターン行動） / パッシブ効果のみ / スキル提供のみ 高
仲間と職業の関係 仲間ごとに固有ステータス・スキルを持たせる（仲間＝職業）/ 仲間と職業を別システムにする 高
主人公の扱い 常にパーティ固定 / 仲間の1人として交代可能 中
仲間の解放方法 イベント / 実績 / ショップ / フロアクリア報酬 など 中
仲間の死亡ルール 戦闘中に倒れても続行 / 倒れたらゲームオーバー / 蘇生アイテムで復活 中
ランをまたいだ引き継ぎ 仲間を一度解放したら次ランも使える（ローグライト）/ 毎ラン最初から集め直す（ローグライク） 中
仲間の総数・実装スコープ 初期実装で何人用意するか（多いほど実装コスト大） 低

③ レリック・装備システム

議題 選択肢・論点 優先度
レリックの帰属 パーティ共有（現状）/ キャラ個別装備 / 共有＋個別の2層構造 高
装備システムの追加有無 キャラごとに装備スロットを作りステータスアップ専用にする / レリックだけで完結させる 高
装備とレリックの役割分担 装備＝シンプルな数値増加、レリック＝ビルド方向性の決定　という分担が考えられる 中
装備の入手方法 ショップのみ / 戦闘ドロップ / イベント報酬 など 低

④ バトルシステム拡張

議題 選択肢・論点 優先度
パーティ戦闘時のターン構造 全員が毎ターン行動 / 1ターンに1人だけ行動を選ぶ / 素早さ順に自動決定 高
全滅条件 全員HP0でゲームオーバー / 主人公が倒れたらゲームオーバー / 別条件 中
敵の複数体対応 パーティ導入に合わせて敵も複数体出現させるか 中
スキルのターゲット選択 敵が複数になった場合、全体攻撃・単体攻撃の区別が必要になる 低

⑤ 短期的な改善（大きな設計変更なしにできること）

議題 内容 優先度
ユーザーフィードバックの収集 審査員のアドバイス通り、実際に触ってもらえる環境を作る（公開URL・共有方法） 高
バランス調整 現状のレリック79個・スキル15種・イベント31個のバランスを実プレイで確認する 高
BGM・SE の追加 ゲーム体験の完成度を上げるために効果音・BGMを検討する 中
スマホ対応 タップ操作・レイアウト調整が必要、ユーザー層が大きく広がる 中
