2. ファイルと担当の詳細

js/shop.js 担当 A
ショップ全般の処理。タブ切り替え・購入済み判定・購入処理（武器/防具/スキル/アイテムで処理が異なる）を担当。

主な関数: initShop() / renderShop() / isOwned() / doBuy()
data/relics.js 担当 A
武器4種・防具4種のデータ定義。攻撃/防御ボーナス・MPボーナス・価格を管理。
今後：レリックシステムへの改変予定。

主な定数: WEAPONS / ARMORS
js/battle.js 担当 B
バトル全般。敵のスポーン・プレイヤー行動・敵ターン・勝敗判定・ダメージ計算・Canvasスプライト描画（11種）を含む最大ファイル。

主な関数: spawnEnemy() / doAttack() / doEnemyTurn() / drawDragon() 等
data/enemies.js 担当 B
10種の敵データ（スライム〜ドラゴン）と階層別出現テーブルを管理。HP・攻撃・防御・報酬ゴールド・ボススキルなど。

主な定数: ENEMIES / FLOOR_ENEMIES
data/items.js 担当 B
6種の消耗品定義。type フィールドでバトル中の使用効果が変わる。
ポーション / ハイポーション / エーテル / 爆弾 / 毒の小瓶 / 蘇生草

主な定数: ITEMS
js/events.js 担当 C
イベントデータと処理を追加していくファイル。現在はプレースホルダー。 EVENTS 配列にイベントオブジェクトを追加する。

主な変数: EVENTS（配列）　主な関数: initEvent()
data/skills.js 担当 C
6種のスキル定義とショップ販売スキル一覧。MPコスト・タイプ（物理/魔法/回復/バフ）・威力を管理。

主な定数: SKILLS / SHOP_SKILLS
js/game.js 共　通
ゲームのコア。全ファイルから参照する GS オブジェクト・シーン管理・フロア進行・スコア計算・ユーティリティ関数を担当。

主な要素: GS / showScene() / initFloorSelect() / pct()
