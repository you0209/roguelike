# CHANGELOG

## [v0.8.0] - 2026-04-05

### 追加
- ダンジョンを7階層に拡張（雑魚戦6階層 + ボス戦1階層）
- 4〜6階層の新敵9種を追加（新規ドット絵スプライト）
  - 4階：ゴーレム / ヴァンパイア / リザードマン
  - 5階：デーモン / メデューサ / ワイバーン
  - 6階：デスナイト / アークメイジ / キマイラ
- 4〜6階層にプレースホルダーイベントを追加（「イベントが起きました」表示）

### 変更
- ボス戦を第4階層 → 第7階層に移動
- ボス（龍神ヴァルムドラグ）のステータスを強化
  - HP: 320 → 500、ATK: 48 → 65、DEF: 28 → 38、報酬: 300G → 500G

## [v0.7.0] - 2026-04-05

### 追加（担当A）
- レリックシステムの導入
  - 武器・防具を廃止し「レリック」に統合（複数所持可、上限なし）
  - レリック8種：鉄の剣 / 鋼の剣 / 炎の剣 / 革の鎧 / 鉄の鎧 / 生命の指輪 / 魔法のローブ / 竜鱗の欠片
  - tier1〜3の重み付けによりレア度を設定
- ショップシステムの刷新
  - タブを「レリック」「アイテム」の2つに統合（スキルタブ廃止）
  - 入店ごとにレリック5個＋アイテム3個をランダム抽選（ラインナップが毎回変わる）
  - 価格を全体的に引き下げ、購入しやすく調整

### 追加（担当C）
- イベントシステムの本実装
  - 階層別イベントプール（各階層5〜6種類）
  - 各イベントに選択肢を設置、選択結果をその場で表示
  - 実装済みイベント一覧：
    - 1階層：宝箱 / 守護の気配（防御バフ） / 運命の輝き（ステータス強化 or 弱体化） / 戦士の覚醒（攻撃バフ） / 罠宝箱
    - 2階層：宝箱 / 血の盟約（HP削りでゴールド2倍） / 強敵の予感（敵強化+報酬2倍） / 魔力解放の石（MP消費なし） / 落とし穴宝箱 / 黄金の盾
    - 3階層：宝箱 / 覇者の契約（ゴールド消費で全ステ2倍） / 奇妙な商人（アイテム交換） / ミミック宝箱（戦闘突入） / 富の力（ゴールド量で攻撃倍率UP）
  - `oncePerFloor` フラグで1階層1回限りのイベントを制御
  - イベントプールが空のときは戦闘にフォールバック

### 変更（担当C）
- `GS.atkTotal` / `GS.defTotal` に階層バフ・イベントバフの乗算を追加
- `descendFloor()` に `skipShop` オプションを追加（落とし穴イベント用）
- プレイヤーにイベントバフ用フィールドを追加（`atkBuffMult`, `defBuffMult`, `floorAtkMult`, `floorDefMult`, `goldDouble`, `mpFree`, `goldShield`, `goldPowerActive` など）

## [v0.5.0] - 2026-04-04

### 変更
- ファイルを機能・担当ごとに分割（チーム作業対応）
  - `js/data.js` を削除し、以下のファイルに分割
  - `data/enemies.js`：敵データ・階層別敵リスト（担当B）
  - `data/skills.js`：スキルデータ・ショップスキル一覧（担当C）
  - `data/items.js`：アイテムデータ（担当B）
  - `data/relics.js`：武器・防具データ（担当A、レリック化予定）
  - `js/battle.js`：バトルロジック・スプライト描画（担当B）
  - `js/shop.js`：ショップロジック（担当A）
  - `js/events.js`：イベントデータ・処理（担当C、現在プレースホルダー）
  - `js/game.js`：ゲームコア・シーン管理のみに整理
